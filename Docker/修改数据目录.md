## Docker 修改数据目录

```bash
# 停止 docker 服务
systemstl stop docker.service

# 可选操作
# 移动, 软链旧路径
mv /var/lib/docker /home/
ln -s /home/docker/ /var/lib/

# 修改 /etc/docker/daemon.json 文件
# 修改数据目录
{
  "data-root": "/opt/docker/data"
}

# 启动 docker 服务
systemstl start docker.service

# 验证是否成功, 验证 Docker Root Dir 的值
docker info
```
