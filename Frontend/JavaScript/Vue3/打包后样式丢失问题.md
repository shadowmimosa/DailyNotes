## vue 打包后 components 组件样式丢失问题

_未尝试_

### 参考

1. 怀疑 Webpack 打包代码，未找到问题。
2. 逐一排除，三个入口文件相似，无差异。
3. 发现有问题分支的 router 文件中，所有页面采用了路由懒加载。

**解决方案:** 将有问题分支的页面改为同步加载方式

```js
// 路由懒加载
const AgentAdmin = () => import('pages/agent/agentAdmin.vue')

// 改为同步加载
import AgentAdmin from 'XXXXXXXXXX'
```
