## Vue 中使用空格占位符

- 使用 v-html
  ```html
  <div v-html="'你好&nbsp;&nbsp;&nbsp;你好'"></div>
  ```
- 使用 pre 标签
  pre 元素可定义预格式化的文本, 被包围在 pre 元素中的文本通常会保留空格和换行符
  而文本也会呈现为等宽字体
  ```html
  <pre>你好&nbsp;&nbsp;&nbsp;你好</pre>
  ```

### 多种空格占位符

```html
<!-- 中文全角空格, 一个中文宽度 -->
&#12288;

<!-- 一个汉字的间距 -->
&#8195; &emsp; em

<!-- 半个文字的间距 -->
&#8194; &ensp; en

<!-- 四分之一个汉字的间距 -->
&#8197; 四分之一em

<!-- 普通的英文半角空格 -->
&#32;

<!-- 普通的英文半角空格但不换行 -->
&#160; &nbsp; &#xA0; no-break space

<!-- 不间断空格 -->
\u00A0 \xa0

<!-- 全角空格(中文符号) -->
\u3000

<!-- 半角空格(英文符号) -->
\u0020
```

相比平时的空格`&#32;`，`&nbsp` 拥有不间断 (non-breaking) 特性
即连续的 `&nbsp` 会在同一行内显示, 即使有 100 个连续的 `&nbsp` 浏览器也不会把它们拆成两行
