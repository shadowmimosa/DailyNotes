## Vue 中使用富文本编辑器

### tinymce

[tinymce 官方文档](https://www.tiny.cloud/docs/)
[tinymce 中文文档](http://tinymce.ax-z.cn/)
[tinymce 中文文档](http://tinymce.ax-z.cn/)
[tinymce-plugins](https://github.com/Five-great/tinymce-plugins)
[demo](https://imageslr.github.io/mweb-themes/)

- 安装

  ```bash
  npm install tinymce @tinymce/tinymce-vue@3.2.8 -S
  ```

- 安装语言包
  [语言包下载](https://www.tiny.cloud/get-tiny/language-packages/)

- 示例
