## Python 中的时间处理

Python 中有两个基础的时间包 `time` & `datetime` & `calendar`  
对于几种时间格式一直没有掌握，导致除了使用基本时间戳以外，每次都要百度，有必要集中总结一下  

### 概述  

- `time`: 主要侧重点在时分秒
- `datetime`: 主要是用来表示日期的，就是我们常说的年月日时分秒
- `calendar`: 主要是用来表示年月日，是星期几之类的信息
