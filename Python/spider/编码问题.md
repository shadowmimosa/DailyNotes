## 爬虫中各种编码问题

### 非常见编码

> 'Accept-Encoding'：br

需要特殊处理

```python
import brotli

key = 'Content-Encoding'
if(key in response.headers and response.headers[key] == 'br'):
    raw = brotli.decompress(response.content)
    content = raw.decode('utf-8')
    print(content)
```
