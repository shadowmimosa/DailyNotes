### Dict 遍历  

1. `clear()` 方法（删除字典内所有元素）  

2. `pop()` 方法（删除字典给定键 key 所对应的值，返回被删除的值）  

3. `popitem()` 方法（随机删除并返回字典中的一堆键值）  

4. `del()` 全局方法（）能删除单一元素也能清空字典，清空只需一项操作）  
```python
dict={key:value}

del dict[key] # 删除键是 key 的条目
del dict # 清空字典所有条目
```
