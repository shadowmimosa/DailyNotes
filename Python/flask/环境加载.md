## Flask 环境变量加载

### 默认加载

通过 `flask run` 启动项目, 不指定 env 时， 默认加载 `.env` 配置

### 手动加载

通过 python-dotenv 包手动加载

```python
from dotenv import load_dotenv

load_dotenv('.flaskenv')
```
