## 远程连接配置

```sql
-- 修改 root 用户的 host 为 %
-- 即 所有 ip 皆可访问，默认为 localhost
use mysql
update user host='%' where user='root';
flush privileges;
```

## 远程连接配置 - Mysql 8.0

- root 用户
  ```sql
  GRANT ALL PRIVILEGES ON *.* TO 'root'@'%'IDENTIFIED BY '密码' WITH GRANT OPTION;
  ```
- 非 root 用户
  ```sql
  -- 切换数据库
  use mysql;
  -- 创建用户
  CREATE USER '用户名'@'%' IDENTIFIED BY '密码';
  -- 授权（默认新创建用户无权限，这里是赋予所有权限）
  grant all on *.* to '用户名'@'%'；
  ```
