## MySql 启动命令

在 CentOS 下可使用如下命令

- 启动

  > service mysqld start 或者
  > /etc/init.d/mysqld start

- 重新启动

  > service mysqld restart 或者
  > /etc/init.d/mysqld restart

- 停止
  > service mysqld stop 或者
  > /etc/init.d/mysqld stop
