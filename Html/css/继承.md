## CSS 继承规则

CSS 样式的继承规则 宽高的继承

- 宽度 子元素对父元素继承(子元素默认 100%宽度)
- 高度 父元素会被子元素撑开
- 与文本相关样式,子元素默认会继承父元素

当固定定位(fixed)和绝对定位(absolute)时, 子元素不会继承父元素的宽度
当父元素为 inline-block 时, 宽度会被子元素撑开
