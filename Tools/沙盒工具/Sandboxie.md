## Sandboxie

前两年即试用了 Sandboxie，当时没用没有，所以一直就在电脑里躺尸了
最近换了新设备，正好有沙盒的需求，Windows 自带的 Sandbox 用后即弃，不能保留现场
稍微看了下 Sandboxie 怎么用，结论是，很好用，超预期

### Sandboxie-Plus
[Github](https://github.com/sandboxie-plus/Sandboxie)

Sandboxie 有两个版本：Plus 和 Classic。它们共享相同的核心组件，这意味着它们具有相同级别的安全性和兼容性。不同之处在于用户界面中功能的可用性。

Sandboxie Plus 具有基于 Qt 的现代 UI，支持自项目开源以来添加的所有新功能：

- 快照管理器 - 获取任何盒子的副本以便在需要时恢复
- 维护模式 - 允许在需要时卸载/安装/启动/停止 Sandboxie 驱动程序和服务
- 便携式模式 - 您可以运行安装程序并选择将所有文件提取到一个目录
- 用于阻止访问 Windows 组件（例如打印机后台处理程序和剪贴板）的其他 UI 选项
- 更多针对启动/运行和互联网访问限制的自定义选项
- 隐私模式沙箱，保护用户数据免遭非法访问
- 安全增强的沙箱，限制系统调用和端点的可用性
- 用于终止所有盒装进程的全局热键
- 每个沙箱都有一个网络防火墙，支持 Windows 过滤平台 (WFP)
- 可以使用快捷键 Ctrl+F 搜索沙箱列表
- 全局设置和沙箱选项的搜索功能
- 能够向 7z 文件导入/导出沙箱
- 将沙箱集成到 Windows 开始菜单中
- 浏览器兼容性向导，用于为不支持的浏览器创建模板
- 复古视图模式可重现 Sandboxie Control 的图形外观
- 故障排除向导可帮助用户解决问题
- 附加组件管理器可通过附加组件扩展或添加功能
