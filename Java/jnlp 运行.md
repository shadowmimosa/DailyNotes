## 运行 jnlp 文件

### 安装 JRE

- 从[oracle](https://www.oracle.com/java/technologies/downloads/)找到合适的版本
- 注意：太高的版本可能无法运行
- 安装
- 配置环境变量

### 运行

使用 bin 目录下的 javaws 运行 jnlp 文件
