- 查看 git 所有配置
    >　git config --list

- 查看 git 全局/本地配置
    > git config --global/--local --list

- 删除全局/本地配置项
    > git config --global/--local --unset user.name/user.email

- 设置 push.default
    > git config --global push.default simple/matching
