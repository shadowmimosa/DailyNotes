## git 推文件上 github 去后显示发送人为 unknown

- 查看配置信息

  > git config -l

  查看 user.name、user.email 是否正确配置

- 修改配置信息

  > \$ git config --global user.name "your_name"

  > \$ git config --global user.email "your_email@youremail.com"

  加上 --global 参数为全局配置
